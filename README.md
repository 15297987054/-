
## 概述

小程序-题库

## 安装

```bash
1. git clone git@gitlab.aihuaiedu.com:aihua-frontend/aihua-question-bank.git

2. cd aihua-question-bank

3. npm i

4. npm run dev
```

## 技术选型

mpVue

## 开发方式

1. 用自己喜欢的编辑器编辑代码.
2. 用小程序开发者工具效果预览.
